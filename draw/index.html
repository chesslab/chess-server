/draw

Allows to offer a draw.

Name Description Required
action propose
decline
accept		 Yes

Usage

Propose a draw

ws.send('/draw propose');

{
  "/draw": {
    "action": "propose"
   }
}

Decline a draw

ws.send('/draw decline');

{
  "/draw": {
    "action": "decline"
   }
}

Accept a draw

ws.send('/draw accept');

{
  "/takeback": {
    "action": "accept"
   }
}
