/draw

Allows to offer a draw.

action

The action to take as per these options.

  • propose
  • decline
  • accept

Usage

Propose a draw

ws.send('/draw propose');

{
  "/draw": {
    "action": "propose"
   }
}

Decline a draw

ws.send('/draw decline');

{
  "/draw": {
    "action": "decline"
   }
}

Accept a draw

ws.send('/draw accept');

{
  "/takeback": {
    "action": "accept"
   }
}
