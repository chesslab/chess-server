/rematch

Allows to offer a rematch.

action

The action to take as per these options.

  • propose
  • decline
  • accept

Usage

Propose a rematch

ws.send('/rematch propose');

{
  "/rematch": {
    "action": "propose"
   }
}

Decline a rematch

ws.send('/rematch decline');

{
  "/rematch": {
    "action": "decline"
   }
}

Accept a rematch

ws.send('/rematch accept');

{
  "/rematch": {
    "action": "accept"
   }
}
